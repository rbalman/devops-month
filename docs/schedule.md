# 30-Day Schedule

Each session is **1–1.5 hours** of focused work: ~20 min theory + ~50 min hands-on lab + take-home assignment.

---

## Week 1 — Foundation

| Day | Topic | Key Skills |
|---|---|---|
| [Day 01](week-01/day-01.md) | DevOps Intro & Machine Setup | What is DevOps, install tools, set up environment |
| [Day 02](week-01/day-02.md) | Linux Basics I | Filesystem, navigation, core commands |
| [Day 03](week-01/day-03.md) | Linux Basics II | Users, permissions, processes |
| [Day 04](week-01/day-04.md) | Linux Basics III | Package management, systemd services |
| [Day 05](week-01/day-05.md) | Bash Scripting I | Variables, conditionals, loops |
| [Day 06](week-01/day-06.md) | Bash Scripting II | Functions, scripts, cron jobs |
| [Day 07](week-01/day-07.md) | Git & GitHub | Commits, branches, pull requests |

## Week 2 — Networking & Web

| Day | Topic | Key Skills |
|---|---|---|
| [Day 08](week-02/day-08.md) | Networking I | IP, ports, TCP/UDP, ping, curl |
| [Day 09](week-02/day-09.md) | Networking II | SSH, DNS, TLS, `dig`, `nmap`, `openssl` |
| [Day 10](week-02/day-10.md) | Nginx | Install, configure, serve static & proxy |
| [Day 11](week-02/day-11.md) | Apache & SSL | Virtual hosts, Let's Encrypt, HTTPS |
| [Day 12](week-02/day-12.md) | Containers I | Docker concepts, `run`, `ps`, `logs`, `exec` |
| [Day 13](week-02/day-13.md) | Containers II | Dockerfile, build, tag, push to registry |
| [Day 14](week-02/day-14.md) | Docker Compose | Multi-container apps, volumes, networks |

## Week 3 — Infrastructure as Code

| Day | Topic | Key Skills |
|---|---|---|
| [Day 15](week-03/day-15.md) | Vagrant | Vagrantfile, provision VMs locally |
| [Day 16](week-03/day-16.md) | Ansible I | Inventory, ad-hoc commands, modules |
| [Day 17](week-03/day-17.md) | Ansible II | Playbooks, tasks, handlers |
| [Day 18](week-03/day-18.md) | Ansible III | Roles, variables, templates |
| [Day 19](week-03/day-19.md) | Terraform I | Providers, resources, `plan` & `apply` |
| [Day 20](week-03/day-20.md) | Terraform II | State, variables, modules |
| [Day 21](week-03/day-21.md) | Review & Mini Project | End-to-end infra with Ansible + Terraform |

## Week 4 — Cloud, CI/CD & Monitoring

| Day | Topic | Key Skills |
|---|---|---|
| [Day 22](week-04/day-22.md) | AWS Basics I | IAM, EC2, key pairs, security groups |
| [Day 23](week-04/day-23.md) | AWS Basics II | VPC, subnets, S3, RDS overview |
| [Day 24](week-04/day-24.md) | CI/CD I | GitHub Actions: triggers, jobs, steps |
| [Day 25](week-04/day-25.md) | CI/CD II | Build Docker image, push, deploy to EC2 |
| [Day 26](week-04/day-26.md) | Monitoring I | Prometheus: scrape, metrics, exporters |
| [Day 27](week-04/day-27.md) | Monitoring II | Loki: log collection, LogQL basics |
| [Day 28](week-04/day-28.md) | Grafana & Alerts | Dashboards, alert rules, Discord webhook |
| [Day 29](week-04/day-29.md) | Final Project | Full end-to-end deploy with monitoring |
| [Day 30](week-04/day-30.md) | Job Preparation | Resume, interviews, what to build next |

---

!!! tip "Pace yourself"
    Each day builds on the previous. If you fall behind, catch up the same day — don't carry debt into the next session.
