# 30-Day Schedule

Each session is **1–1.5 hours** of focused work: ~20 min theory + ~50 min hands-on lab + take-home assignment.

---

## Week 1 — Linux System Administration

| Day | Topic | Key Skills |
|---|---|---|
| [Day 01](week-01/day-01.md) | DevOps Intro & Machine Setup | What is DevOps, install tools, set up environment |
| [Day 02](week-01/day-02.md) | Linux History & Filesystem | Origins, open source, FHS, key directories |
| [Day 03](week-01/day-03.md) | Everything Is a File | Streams, `cat`/`grep`/`sed`/`awk`/`cut`/`tr`, pipelines |
| [Day 04](week-01/day-04.md) | Users, Groups & Permissions | `chmod`/`chown`, special bits, ACLs, file attributes |
| [Day 05](week-01/day-05.md) | The Shell | Shell types, profiles, env vars, exit codes, aliases, functions |
| [Day 06](week-01/day-06.md) | Shell Scripting I | Variables, input, operators, command substitution |
| [Day 07](week-01/day-07.md) | Shell Scripting II | Conditionals, loops, functions, arguments, cron |

*See also: [Beyond the Basics](week-01/beyond-the-basics.md) — a reference map of Linux topics, config files, and commands we don't cover in class.*
